Application Documentation

Welcome to the Address Cleaning Form documentation. This application helps you validate and correct address inputs by ensuring that data entered is not misspelled and country names are standardized.

How It Works

  • Data flow:First it gusses the standardized country name and ISO code, then it trys to guess the state that may be entered and then city
  • How it gusses:It uses the GeoNames API to get the list of things within the world/country/state and then it finds the words from the API data that has the least edit distance with respect to the word received from the user.
  • Searching constraint:In order to find the least edit distance, we require a search space from which we pick the relevant words. That requires us to give some direction to search query from the knowledge graph because for example there can be thousands of cities in a state. So in this case, I first fetch the cities/ states whose first 3 letters match the entered city/ state. This is not the case for countries are they are relatively limited.
